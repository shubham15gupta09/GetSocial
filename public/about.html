Hello User , Glad that you want to know more about Get Social

Get Social is a social media platform initially made by Mr. Shubham Gupta who is a 4th year Btech Student . He has open sourced this project so that more features can be added by other Users

All the Data are stored in India only using MongoDb free tier cluster created using Google Cloud Platform (GCP) on Mumbai (asia-south1)

Features that will be included soon :-

  • Profile card
  • Block / Unblock a User
  • Profile card
  • Like / Retweet a post
  • Sign in with third party platform like Google , Twitter
